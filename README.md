# QR Code Generator

This is a simple QR code generator written in Python. It allows you to generate QR codes for various types of data, such as URLs, text, contact information, and more.

## Features

- Generate QR codes for URLs, text, contact information, and more.
- Customizable QR code size, error correction level, and background/foreground colors.
- Save generated QR codes as image files (PNG format).

## Requirements

- Python 3.x
- `qrcode` library
